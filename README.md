# Exercícios JavaScript 2025

Este repositório contém exercícios práticos desenvolvidos durante o ano de 2025 na disciplina de Web Designer.

## Sobre os exercícios

As atividades foram voltadas à prática da integração das tecnologias de desenvolvimento front-end:

* HTML → Estrutura das páginas
* CSS → Estilização
* JavaScript → Lógica e interações básicas

## Organização

O repositório está dividido em 10 exercícios independentes, cada um contendo seus próprios arquivos:

* `index.html`
* `style.css`
* `script.js`

## Objetivo

Praticar conceitos introdutórios de programação em JavaScript aplicados a páginas web, utilizando sua integração com HTML e CSS nas atividades propostas em aula.
